Add a feature idea to the **Future Feature Ideas** section in `docs/ROADMAP.md`.

**Input:** $ARGUMENTS

## Instructions:
1. Read `docs/ROADMAP.md`
2. Find the "Future Feature Ideas" section (create it above the final `---` if it doesn't exist)
3. Generate a sequential idea ID (IDEA-001, IDEA-002, etc.) by checking existing IDs
4. Add the new idea as a bullet point with the ID and today's date
5. Keep it concise - just the idea, not a full spec

## Format:
```md
### Future Feature Ideas

- **IDEA-001** | 2025-12-16 | Description of the idea
- **IDEA-002** | 2025-12-16 | Another idea
```

Show the feature ID after adding it.
